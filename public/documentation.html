myFlix Documentation

Request Method URL Response
Get All Movies GET localhost:8080/movies This request returns a JSON object containing all of the movies and all of their respective data.
Get Movie by Title GET localhost:8080/movies/:title This request returns a JSON Object of a single movie in the form of an array
Get Genre Data by Name GET locahost:8080/movies/genre/:genre This request returns a JSON Object that contains information about a single genre, including the genre name and a short description
Get Director by Name GET locahost:8080/movies/directors/:directorName This request returns a JSON Object that contains information about one director, including his name, a short biography, and his year of birth
Create New User POST locahost:8080/users This request will create a new user with a unique user ID and return the new user object
Update Usernames PUT locahost:8080/users/:id This request will update the username of user, provided the given username already exists to replace
Add to User's Favorite Movie List POST locahost:8080/users/:id/:movieTitle This request will update the given user's favorite movie list with the provided movie, given the user id exists. It will return a message to confirm the movie has been added successfully
Remove from User's Favorite Movie List DELETE locahost:8080/users/:id/:movieTitle This request will remove the specified movie from the provided user's favorite movie list, given the user id exists. It will return a message to confirm the movie has been added removed
Remove User DELETE locahost:8080/users/:id This request will remove the specified user from the user's array if the user exists in the array. It will return a message to confirm the user has been deleted.
